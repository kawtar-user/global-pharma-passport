# Global Pharma Passport Web

## Run locally

```bash
cd /Users/kawtar/Documents/New\ project\ 3/apps/web
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000), then visit `/dashboard`.

## Current scope

- login page
- signup page
- patient dashboard
- medication add form
- interaction alerts panel
- international equivalents table
- travel mode button
- travel passport preview

The current UI uses realistic mock data from `src/lib/dashboard-data.ts` so we can iterate on product and design before wiring the live API.
